# proxy
Python爬虫代理IP池(proxy pool)

# 项目依赖
Spracy框架 丶 Flask框架

# Api接口
1. ip接口响应为JSON格式 http://127.0.0.1:5000
2. 删除ip接口 http://127.0.0.1:5000/delete?ip=120.92.3.127
